글자 쌓기

1. 문자열을 id가 'inputText'인 input창에 입력받습니다. 입력 버튼을 누르면 아래의 지시사항을 수행한다.
2. 각 글자를 1x1 크기의 블록이라고 생각하고 띄어쓰기 단위로 하나의 블록으로 묶어준다.
ex) '안녕하세요 반갑습니다'는 5x1 크기의 블록이 2개이다.
3. 블록을 순서대로 세로로 나열한 뒤 나열된 블록에 겹쳐지는 가장 큰 크기의 정사각형을 찾는다.
4. 'resultBox'라는 id를 가진 'div' 태그에 'block'이라는 class를 가진 div 태그를 만들어 채워넣는다.
ex) 3번에서 구한 정사각형이 3x3 크기라면 총 9개의 div 태그를 3x3모양으로 채워넣는다. (display: flex로 배치한다.)
5. 다른 값을 입력하고 버튼을 누르면 기존의 div태그들을 삭제하고 새로운 결과를 출력한다.
6. 공백을 입력하면 박스를 비운다.

Test Case1: 안녕 나는 엘리스야 / 2x2

Test Case2: 안녕하세요 존 반가워요 / 1x1

Test Case3: Hello World this is Amazing / 3x3

문제 예시
입력값: '안녕하세요 반갑습니다 오늘은 배운 내용이 어렵네요'
