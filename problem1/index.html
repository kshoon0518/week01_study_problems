팩토리얼 계산기

1. 1이상 15 이하의 수를 input창에 입력받습니다.
2. 입력받은 수가 정수가 아니라면 반올림을 한다.
3. 자바스크립트를 통해서 id가 'btn'인 버튼을 누르면 입력한 숫자의 팩토리얼 결과를 계산한다.
4. id가 'result'인 'div'에 결과값을 출력한다.
5. 입력받은 숫자가 1이상 15이하가 아닌 경우 공백을 표시합니다.

Test Case1: 0.9 / 공백

Test Case2: 1.9 / 2

Test Case3: 5.5 / 720

Test Case4: 14.9 / 1307674368000
